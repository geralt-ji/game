# Flappy Bird 克隆版 (版本 1.0)

版本日期：2024-10-11

这是一个使用 Python 和 Pygame 库创建的 Flappy Bird 风格的平台跳跃游戏。

## 版本 1.0 特性

- 基本的 Flappy Bird 游戏机制
- 可调整窗口大小
- 无敌模式（加速功能）
- 暂停功能
- 关卡系统
- 随机生成的管道
- 分数计算
- 简单的碰撞检测

## 游戏说明

在这个游戏中，玩家控制一个小方块（代表小鸟），通过点击空格键或鼠标左键使小方块跳跃。目标是在不碰到管道的情况下，尽可能长时间地保持飞行。

## 安装说明

1. 确保你的系统中安装了 Python 3.x
2. 安装 Pygame 库：
   ```bash
   pip install pygame
   ```
3. 下载或克隆这个仓库
4. 运行 `main.py` 文件开始游戏

## 控制

- 空格键或鼠标左键：使小方块跳跃
- ESC 键：退出游戏
- P 键：暂停/继续游戏
- 点击左下角的 "God" 按钮：切换无敌模式

## 文件结构

- `main.py`: 主游戏循环和初始化
- `game.py`: 游戏逻辑
- `bird.py`: 小方块（小鸟）类
- `pipe.py`: 管道类
